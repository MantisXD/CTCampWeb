CT Camp 저장 웹 개발
2017 OOP Project로 CT Camp 등의 여러 자료들이 한번 개발 된 뒤에 바로 사라지거나 잊혀지는 것이 아쉬워 이것들을 저장하는 페이지를 개발했다. DB, WEB Application을 사용할까도 했지만 학기 중에 시간도 부족하다 생각하여 찾아본 결과 Static site로 개발하기로 생각하였다. 실제로 연간 몇회 수정이 없을 예정의 사이트로 생각되기에 static site로 구현하게 되면 갖는 이점이 많다. 유지보수도 편리하고 코스트도 적게 들면서 보안적으로도 안전하다.
